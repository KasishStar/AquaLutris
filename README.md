# AquaLutris

Modern macOS-inspired game launcher.

AquaLutris aims to bring a clean and modern launcher experience inspired by Lutris and macOS design principles.

## Features

* Launch AppImages
* Game Library
* Favorites
* Cover Artwork
* Search
* Runner System
* Modern Aqua UI
